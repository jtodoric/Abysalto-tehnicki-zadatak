<h1 align="center">Automatizacija Web Trgovine (Playwright)</h1>

Ovaj repozitorij sadrži rješenje za automatizaciju E2E (End-to-End) procesa kupovine na testnoj stranici automationpractice.pl. Skripta pokriva registraciju korisnika, odabir proizvoda, dodavanje u košaricu, kontaktiranje podrške i završetak procesa narudžbe.
